# COBOL Batch Processing

This project simulates a batch processing system developed in COBOL, similar to those used in mainframe environments.

## Features
- Reads an input file
- Processes records
- Generates an output file

## Technologies
- COBOL
- File handling
- Batch processing

## Purpose
This project demonstrates experience working with legacy systems and batch processes, commonly used in banking and large-scale data environments.

## Notes
This is a basic example intended to showcase core COBOL and batch processing concepts.
